# IT Park Zomin - IT Shaharchasi

> Zamonaviy IT ta'lim markazi - Kelajakni birga quramiz

## 🌐 Haqida

IT Park Zomin tuman filiali - Zomin shahridagi zamonaviy axborot texnologiyalari markazi. Biz yoshlarni dasturlash, web-dizayn, mobil dasturlash, UI/UX dizayn va boshqa IT sohalarida professional mutaxassislar sifatida tayyorlaymiz.

### 🎯 Missiyamiz

- Zamonaviy IT ta'limini ommalashtirish
- Yoshlarni raqamli iqtisodiyotga tayyorlash
- Professional IT mutaxassislar yetishtirish
- Zomin tumani va Jizzax viloyatini IT markaziga aylantirish

## 🚀 Xususiyatlari

### 📚 Kurslar
- **Web Dasturlash** - Frontend va backend texnologiyalari
- **Mobil Dasturlash** - Android va iOS ilovalar
- **UI/UX Dizayn** - Foydalanuvchi interfeysi va tajribasi
- **Grafik Dizayn** - Vizual kommunikatsiya va brending
- **Kiberxavfsizlik** - Tarmoq himoyasi va xavfsizlik
- **Data Science** - Ma'lumotlar tahlili va AI

### 🎨 Dizayn
- Zamonaviy va responsive interfeys
- Dark/Light theme rejimlari
- Animatsiyalar va transition effektlar
- Mobil qurilmalar uchun optimallashtirilgan

### ⚡ Texnologiyalar
- **Frontend**: React 18, TypeScript, Tailwind CSS
- **Build Tool**: Vite
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **UI Components**: Radix UI + shadcn/ui

## 🛠️ O'rnatish

### Talablar
- Node.js 18+ 
- npm yoki yarn

### Qadamdar

1. **Repositorni klonlash**
```bash
git clone https://github.com/xusanbek0039/itclms.uz.git
cd itclms.uz
```

2. **Paketlarni o'rnatish**
```bash
npm install
# yoki
yarn install
```

3. **Development serverni ishga tushurish**
```bash
npm run dev
# yoki
yarn dev
```

4. **Browserda ochish**
```
http://localhost:5173
```
