# SEO Optimizatsiya Tavsiyalari - IT Park Zomin

## 🚀 Amalga Oshirilgan SEO Yaxshilanishlar

### 1. Meta Teglar
- ✅ Kengaytirilgan kalit so'zlar ro'yxati (50+ ta IT sohalari)
- ✅ Optimallashtirilgan meta description
- ✅ Geo-lokatsiya teglari (Zomin, Jizzax viloyati)
- ✅ Yandex verification meta tegi
- ✅ Qo'shimcha SEO teglari (language, distribution, rating)

### 2. Open Graph & Twitter Cards
- ✅ To'liq Open Graph teglari (title, description, image, image:alt)
- ✅ Twitter Card teglari (summary_large_image)
- ✅ Ijtimoiy media uchun optimallashtirilgan tasvir o'lchamlari

### 3. Structured Data (JSON-LD)
- ✅ EducationalOrganization schema
- ✅ Kurslar katalogi (Web dasturlash, Mobil dasturlash, UI/UX, Grafik dizayn)
- ✅ Geo-coordinates va manzil ma'lumotlari
- ✅ Aloqa ma'lumotlari va tillar
- ✅ Ijtimoiy media havolalari

### 4. Sitemap & Robots.txt
- ✅ XML sitemap (barcha bo'limlar va anchor linklar)
- ✅ Robots.txt (Google, Yandex, Bing, Twitter, Facebook)
- ✅ Crawl delay sozlamasi

### 5. Semantic HTML5 & Accessibility
- ✅ Semantic teglar (header, main, nav, section)
- ✅ ARIA teglari va accessibility yaxshilanishlari
- ✅ Alt text optimallashtirish

## 🔍 Qidiruv Tizimlari Uchun Tavsiyalar

### Google Search Console
1. [Google Search Console](https://search.google.com/search-console) ga ro'yxatdan o'ting
2. Saytni tasdiqlang (DNS yoki HTML fayl orqali)
3. Sitemapni qo'shing: `https://itclms.uz/sitemap.xml`
4. Index status va qidiruv tahlillarini kuzating

### Yandex Webmaster
1. [Yandex.Webmaster](https://webmaster.yandex.ru) ga ro'yxatdan o'ting
2. Saytni tasdiqlang
3. Yandex verification meta tegini yangilang:
   ```html
   <meta name="yandex-verification" content="ACTUAL_YANDEX_CODE" />
   ```

#### 3. Texnik SEO

#### 4. Link Building
- Mahalliy ta'lim muassasalari bilan hamkorlik
- IT forumlar va guruhlarda ishtirok
- Guest posting (boshqa IT saytlarida)
